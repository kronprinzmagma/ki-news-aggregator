# Build-Anker

Baue mit der Claude API einen minimalen Zitatprüfer: Übergib einen Textabschnitt mit Quellenangaben, lass Claude jeden Beleg gegen den zitierten Inhalt prüfen und gib eine Liste nicht verifizierbarer Stellen aus – so siehst du, wo ein Reviewer-Layer echten Wert schafft.

# Build-Anker

Lies mit `uv tree --outdated` alle veralteten Pakete aus einem bestehenden Projekt aus, trage danach für `pydantic`, `torch` und `transformers` manuell `>=x.y,<x+1`-Schranken in `pyproject.toml` ein und dokumentiere, welche der ursprünglich angezeigten Updates danach blockiert werden.

# Build-Anker

Baue dasselbe lokal nach: Ruf die Wikipedia-API (`https://en.wikipedia.org/api/rest_v1/page/html/{Seitenname}`) mit Python und `requests` ab, parse alle `<table>`-Tags mit `BeautifulSoup` und sieh, wie viele strukturierte Tabellen eine beliebige Wikipedia-Seite enthält – vergleiche das Ergebnis für drei verschiedene Seiten.

# Build-Anker

Setze `CLAUDECODE=1` in deiner Shell und ruf zwei identische `hf`-Befehle auf – einmal ohne, einmal mit gesetzter Variable – und zähle die Zeichen im jeweiligen Output, um den Token-Unterschied direkt zu messen.

# Build-Anker

Öffne Claude.ai, wähle einmal Opus 4 und einmal ein älteres Modell wie Sonnet 3.5, schicke beiden denselben Prompt mit einem klar definierten Werkzeug-Schema (z. B. `path` + `edits[]` wie im Artikel) und dokumentiere per Screenshot, welches Modell sich an die vorgegebenen Felder hält – und welches neue erfindet.

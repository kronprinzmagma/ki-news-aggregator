# Build-Anker

Lege für dein nächstes Prototyp-Projekt drei Wochen lang zwei Zählwerte nebeneinander: KI-geschriebene Codezeilen (z. B. via Cursor-Stats) und eine selbstgewählte Outcome-Grösse (etwa: funktionierende Features, gemessene Fehlerrate). Nach drei Wochen siehst du, ob beide Kurven überhaupt korrelieren.

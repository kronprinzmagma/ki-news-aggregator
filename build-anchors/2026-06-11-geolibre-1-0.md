# Build-Anker

Lade GeoLibre im Browser, öffne die DuckDB-SQL-Konsole, und frag einen öffentlichen GeoParquet-Datensatz per URL ab – so siehst du konkret, wie schnell browserbasierte Geodatenabfragen ohne Backend funktionieren.

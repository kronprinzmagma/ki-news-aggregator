# Build-Anker

Lade eine eigene Sprachaufnahme bei einem kostenlosen Online-Transkriptionsdienst (z. B. Whisper.ai oder Otter.ai) hoch, zähle danach manuell alle Füllwörter im Original und vergleiche, wie viele im Transkript fehlen – so siehst du die 60-%-Lücke, die *erm* schliesst, ohne eine Zeile Code zu schreiben.

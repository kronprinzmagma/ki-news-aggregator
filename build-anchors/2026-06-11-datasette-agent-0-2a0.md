# Build-Anker

Baue mit der `ask_user()`-Logik als Vorbild einen minimalen Python-Agenten (LangChain oder direkt mit der Anthropic-API), der vor jeder schreibenden Aktion – z. B. Datei speichern – eine Bestätigungsfrage stellt, und miss, wie viele unerwünschte Schreibvorgänge dadurch abgefangen werden.

# Build-Anker

Installiere Datasette (eine Browser-Oberfläche für SQLite-Datenbanken, per `pip install datasette datasette-apps`), öffne die mitgelieferte `doomql.sqlite` und lass Claude eine HTML-App generieren, die die Pixelfarben des aktuellen Spielbilds jede Sekunde neu abfragt – so siehst du, ob SQL allein einen vollständigen Rahmen liefert oder Lücken entstehen.

# Build-Anker

Schreib ein Python-Skript, das mit `python-nmap` dein lokales Netzwerk nach offenen UDP-Ports scannt und alle antwortenden Geräte auflistet – so siehst du, welche deiner eigenen Geräte ungefragt auf Netzwerkanfragen reagieren.

Installiere Sem via `cargo install --git https://github.com/Ataraxy-Labs/sem sem-cli`, führe `sem diff` und `sem log --json` auf einem eigenen Repo aus und miss, ob Claude Code die geänderten Funktionen ohne Nachfrage korrekt benennt.

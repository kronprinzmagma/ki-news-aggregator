# Build-Anker

Schreib ein Python-Skript, das den Netzwerkverkehr eines lokal laufenden Prozesses mitliest (z.B. mit `mitmproxy`) und alle ausgehenden HTTP-Aufrufe nach Zeichenketten filtert, die wie API-Keys aussehen – so siehst du sofort, ob ein Tool Keys nach aussen sendet.

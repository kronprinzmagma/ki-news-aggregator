# Build-Anker

Nimm eine eigene kleine Python-Funktion oder CLI, lass Claude Code dieselbe Aufgabe dreimal lösen – einmal mit voller Doku, einmal nur mit Funktionssignatur, einmal ohne jede Erklärung – und zähle jeweils die Token pro Durchlauf: so siehst du konkret, wie viel deine Dokumentation einen Agenten kostet oder spart.
